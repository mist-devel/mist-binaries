This directory contains test versions of the MIST firmware.

These versions aren't tested in every detail, so be prepared to encounter problems.

To install an update rename it to firmware.upg, copy it to your SD card and use the OSD (either atari st or amiga/minimig) to start the update.

For more details about upgrading the firmware read:
http://code.google.com/p/mist-board/wiki/HowToInstallTheFirmware
