# Atari 5200 core for MiST by Mark Watson

## Controls

- Joystick button A - FIRE
- Joystick button B - TRIGGER
- Joystick button C - CPU HALT
- F1 - START
- F2 - PAUSE
- F3 - RESET
- Keypad:
```
123
qwe
asd
zxc
```
